# 付録2：テンプレート・フォーマット集

この付録は、本文で登場する「再現可能なコミュニケーション」を支えるテンプレート集である。自分のチームの運用に合わせて、項目の追加・削除をしてよい。

## T1. Context Package（非同期議論の文脈パッケージ）

```markdown
【Topic】（議論テーマ）

【Objective】（目的 / 成功条件）

【Background】（背景 / これまでの経緯）

【Constraints】（制約: 期限/コスト/品質/運用/法務 等）

【Options】（選択肢: A/B/C）
- A:
- B:
- C:

【Recommendation】（提案）

【Decision Needed】（何を決めたいか / 誰が決めるか）

【Deadline】（いつまでに）

【Links】（一次情報リンク: チケット/PR/ログ/仕様/図）
```

## T2. 会議アジェンダ（45分の意思決定会議）

```markdown
【目的】（何を決めるか）
【決めないこと】（範囲外）
【参加者】（決定者/提案者/レビュー/オブザーバ）

0) 事前共有（会議前）
- Context Package
- 参考資料リンク

1) 文脈共有（5分）
- 前提/制約/成功条件の確認

2) 現状報告（10分）
- 事実とデータ（観測ログ/状況）

3) 論点整理（15分）
- 論点1
- 論点2
- 論点3

4) 意思決定（10分）
- 決定事項
- 採用しない案（理由）

5) 次アクション（5分）
- ToDo（担当/期限）
- 次回までに必要な準備
```

## T3. 議事録（決定事項が残る形）

```markdown
【会議名】
【日時】
【参加者】

【決定事項】
- D1:
- D2:

【保留（宿題）】
- H1:（担当/期限）
- H2:

【リスク/論点】
- R1:

【次アクション】
- A1:（担当/期限）
- A2:

【関連リンク】
- チケット:
- PR:
```

## T4. 非同期依頼（レビュー/判断/作業）

```markdown
【依頼】（一言で）
【目的】（何のために）
【依頼内容】（成果物の形式・粒度）
【期限】（いつまでに / なぜその期限か）
【前提】（制約・背景）
【見てほしい観点】（2〜3点）
【リンク】（一次情報）
【返信の期待】（返信必要/不要、いつまでに）
```

## T5. PRレビュー依頼（観点付き）

```markdown
## 変更の概要
- 何を変えたか
- なぜ変えたか

## 影響範囲
- 影響する: 
- 影響しない:

## リスク
- 想定リスク:
- ロールバック手順:

## テスト
- 実施内容:
- 結果:

## レビュワーに見てほしい点
- 観点1:
- 観点2:
- 観点3:
```

## T6. 仕様変更提案（RFC）

```markdown
## 概要
（一言で）

## 背景
（なぜいま必要か）

## 課題
（現状の問題）

## 提案
（何をどう変えるか）

## 選択肢
- A:
- B:
- C:

## 影響
- スコープ:
- 納期:
- 品質:
- 運用:
- セキュリティ:

## 移行/ロールアウト
（段階移行/後方互換/切り戻し）

## 未決事項
- Q1:
- Q2:
```

## T7. ADR（Architecture Decision Record）

```markdown
# ADR-XXXX: （タイトル）

## ステータス
提案中 / 承認 / 却下 / 廃止

## 背景
（前提と動機）

## 決定
（何を採用するか）

## 代替案
（検討したが採用しない案）

## トレードオフ
（何を得て、何を捨てるか）

## 影響
（実装/運用/コスト/組織）
```

## T8. インシデント連絡（速報/続報/終報）

```markdown
【速報】（発生直後）
- 発生時刻:
- 事象:
- 影響範囲:
- 現在の対応:
- 次回更新:

【続報】（状況変化）
- 追加事実:
- 対応状況:
- 見込み:
- 次回更新:

【終報】（復旧後）
- 根本原因:
- 暫定対応:
- 恒久対応:
- 再発防止:
- 学び:
```

## T9. フィードバック（SBI: Situation/Behavior/Impact）

```markdown
【Situation】（いつ/どこで）
【Behavior】（何をした/しなかった）
【Impact】（何が起きた）

【期待】（次はどうしてほしいか）
【支援】（必要なら支援内容）
```

## T10. 1on1アジェンダ

```markdown
## 近況（事実）
- よかったこと:
- 詰まっていること:

## 相談（論点）
- テーマ1:
- テーマ2:

## 次の一手
- アクション:
- 支援が必要なこと:

## 振り返り
- 今日の気づき:
```
