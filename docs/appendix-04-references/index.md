# 付録4：参考文献・引用

この付録は、本書の「引用」「参考」「定量表現（%/倍など）」の扱いを統一するための方針をまとめるものである。

## A4-1. 定量表現（%向上/%削減 等）の方針

- 本書内の定量表現は、原則として **「目標値の例」「仮説」「目安」** のいずれかとして扱う。
  - 断定が必要な場合は、出典（一次情報）を併記する。
- 出典がない数値は、**「実測値」「一般に保証される効果」** と誤解されない表現に統一する。
  - 例: 「効果：60%向上」→「（目標値の例）意思決定速度の向上」
- 環境差が大きい項目（学習効率、ストレス、満足度 等）は、特に断定を避ける。

## A4-2. 引用・参考の方針（最低限）

- **引用**: 主張の根拠として一次情報をそのまま参照する（URL、文書名、該当箇所を明示）。
- **参考**: 手法や用語の背景を示すための参照（必要に応じてURLを付与）。
- **経験則**: 実務経験に基づく観測は、前提（組織規模/開発形態/ドメイン）を併記し、一般化しすぎない。

## A4-3. 参考文献（要確認）

本書で参照している（または参照したい）文献・資料は、今後この付録に集約する。現時点では、本文中に明示的な外部参照が少ないため、必要に応じて追記が必要である。

- 研究/統計（一次情報）: 要確認
- 公的機関のガイドライン: 要確認
- ベンダの公式ドキュメント: 要確認
- 技術組織の公開事例: 要確認
